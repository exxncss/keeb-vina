Setting keyboard (klik ini lagi kalo mau balik ke menu utama)

baca dulu yaa....

nah, sebelum masuk lebih dalem... aku jelasin dulu nih. jadi keyboard ini tuh punya dua mode koneksi, wired sama wireless. wirelessnya itu kebetulan cuma support bluetooth.

setting rgb

keyboard ini tuh ada beberapa mode gitu vin, jadi kamu bisa on off-in atau pilih mode rgb mana yang kamu suka. caranya teken FN+backspace untuk ganti tipe lednya. terus FN+arrow up/down buat kecerahan lednya, sementara FN+arrow left/right buat kecepetan (wave) rgb-nya. kamu bisa pilih sesuai dengan keinginan kamu hehe

setting wireless/wired mode

nah, klo semisal kamu mau coba mode wireless, dibawah keyboardnya ada tombol off/on. kamu pilih on, abis itu pencet FN+R. itu buat aktifin mode wirelessnya ya. klo untuk balik lagi ke mode wired, kamu matiin tombol dibawah keyboardnya ke off. terus abis itu, colok kabel dan... pencet FN+R lagi. saran aku yaa, klo emang vina jarang pake wireless, mendingan gausa dipake aja. soalnya, agak delay juga sih wirelessnya gitu.

F Row

untuk F row (F1-F12), kamu cukup pencet tombol FN+1 untuk F1, dan seterusnya sampe F12. sampe sampe sini paham kah, cantik?

Multimedia Keys

Untuk multimedia keys, vina bisa pake selagi dengerin lagu/youtube gitu. jadi ada bbrp shortcut, diantaranyaa FN+A untuk play/pause lagu. FN+S untuk previos track, FN+D untuk next track. buat volume, vina bisa pake FN+F untuk mute, FN+G untuk volume down, dan FN+H untuk volume up.
